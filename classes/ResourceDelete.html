    Class ResourceDelete<T>

    Fluent builder for deleting a single resource by ID. Provides a chainable API for configuring DELETE /os/{objectStructure}/{id} requests.

    Example

    await client.asset
      .deleteById('12345')
      .withTransactionId('unique-tx-id')
      .suppressEvents()
      .execute();
