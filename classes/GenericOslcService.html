    Class GenericOslcService<T>

    Generic service for interacting with any Maximo OSLC object structure. Use this when you don't have a dedicated service class for a specific object structure.

    Type Parameters

    • T extends OslcResource = OslcResource

      The resource type, defaults to OslcResource.
